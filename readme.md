# **BDT Format**
## **Javascript _number_ to _money string_ generator according to Bangladeshi format**


### **Javascript _number_ to _money string_ generator according to Bangladeshi format**

How to use:

```
const {bdtFormat} = require('@sashakil12/bdtformat')

bdtFormat(152457899)
```

**Warning!**
**It does not support BigInt's**
